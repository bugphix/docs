Bugphix

Capture and monitor detailed error logs with nice dashboard and UI

Get Started →

Laravel and Vue Powered

Bugphix was built in top of powerful Laravel framework and together with Vuejs framework for dashboard.

Simplicity First

Simple dashboard with no unnecessary information to display in able for you to capture and get only the important details

Self host

Bugphix is a self hosted application, for you to control the whole app, with a simple command installer, you can easily setup the application.
